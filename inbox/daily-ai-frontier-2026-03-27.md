# Daily AI Frontier Update — 2026-03-27

- **Gemini 3.1 Flash Live 发布预览（Google DeepMind）**：Google 在 3/26 宣布通过 Live API 提供 Gemini 3.1 Flash Live，重点是更低语音延迟、更强噪声环境任务完成率与更稳的指令遵循。对实时语音 Agent 来说，这代表可用性和产品化门槛进一步下降。
- **Google 发布“有害操纵”评估方法（DeepMind Safety）**：DeepMind 在 3/26 公布关于 AI 有害操纵风险的新研究与评估工具包，并开放研究材料，推动该风险从“概念讨论”走向可测量、可复现实证评估。
- **OpenAI 上线 Safety Bug Bounty（OpenAI Safety）**：OpenAI 在 3/25 启动面向“滥用与安全风险”的公开赏金计划，补上仅靠传统安全漏洞赏金难覆盖的 AI 特有风险面，尤其是 agentic 场景。
- **OpenAI 公开 Model Spec 方法论（OpenAI Research/Safety）**：OpenAI 在 3/25 详述 Model Spec 的设计哲学与迭代机制，强调“可公开审视的行为规范”在训练、评估与治理中的作用，提升对齐框架透明度。
- **Anthropic 启动 The Anthropic Institute（Anthropic）**：Anthropic（3/11）宣布设立新研究机构，聚焦强 AI 带来的社会级挑战（就业、治理、价值对齐等），显示前沿公司正把“技术能力竞争”扩展到“社会治理能力建设”。
