# Daily AI Frontier Update — 2026-03-28

- **Google launches Gemini 3.1 Flash Live (preview):** Google DeepMind released a low-latency multimodal live model in the Gemini Live API aimed at real-time voice/vision agents, with better noisy-environment robustness and instruction-following. **Why it matters:** This lowers the barrier for production-grade, conversational-speed AI agents.
- **Gemini app March Drop expands cross-product intelligence:** Google announced chat-history migration from other providers, wider free access to Personal Intelligence in the U.S., and stronger Gemini Live context continuity. **Why it matters:** Consumer AI lock-in dynamics are shifting toward easier switching and deeper ecosystem integration.
- **Lyria 3 Pro expands generative music capabilities:** Google introduced up-to-3-minute music generation with better song-structure control, and rolled it into Vertex AI, AI Studio/Gemini API, Google Vids, and Gemini app surfaces. **Why it matters:** Music generation is moving from demos to broader creator and enterprise workflows.
- **OpenAI updates ChatGPT shopping via ACP expansion:** OpenAI product notes report richer visual comparison shopping and expanded merchant connectivity through the Agentic Commerce Protocol. **Why it matters:** Agentic commerce is becoming a serious battleground for AI assistants beyond Q&A.
- **Anthropic Claude Code 2.1.85 release notes highlight dev workflow upgrades:** Latest notes include MCP/header improvements, hook conditionals, improved large-session performance, and many remote/terminal fixes. **Why it matters:** Tooling stability and integration depth continue to be core differentiators in agent IDE workflows.
